# Epic 004: Communication System

## Epic Goal
Implement internal messaging and notification system for communication between organisation levels. This enables coordinated communication for public health officials across the organisational hierarchy, supporting urgent alerts and routine messaging.

## Epic Description
This epic provides comprehensive internal communication capabilities. Users can send messages to individuals or groups, receive notifications, and handle urgent alerts. The system supports organisation-level broadcasting and read receipts for important messages.

## User Stories
- US-022: Internal Messaging
- US-023: Organisation-Level Messaging
- US-024: Notifications
- US-025: Urgent Alerts

## Priority
**Medium** - Important for coordination

## Dependencies
- Epic 001: User & Organisation Management

## Status
Ready for implementation
