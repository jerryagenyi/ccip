# Epic 002: Activity Tracking & Reporting

## Epic Goal
Implement comprehensive activity tracking system with evidence uploads, customisable templates, and audit trails for risk communication activities. This is the core feature of CCIP, enabling public health officials to record, track, and report on risk communication activities with supporting evidence.

## Epic Description
This epic provides the core activity tracking functionality for CCIP. Users can create activities, upload evidence, submit for approval, and track activity status through a complete workflow. Activities support multiple file types, custom templates, and comprehensive audit trails.

## User Stories
- US-005: Create Activity
- US-006: Upload Evidence
- US-007: Activity Submission
- US-008: Activity Timeline
- US-009: Activity Templates

## Priority
**High** - Core feature of CCIP
