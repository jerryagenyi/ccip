## Getting Started Today

### 1. Create Your Knowledge Base
```bash
mkdir -p .bmad/bmm/data/organization
# Create org-context.md, civictech-context.md, team-context.md
```

### 2. Configure Your Agents
```bash
/bmad:bmm:agents:pm "set personality to strategic nonprofit leader..."
/bmad:bmm:agents:analyst "set personality to social impact analyst..."
/bmad:bmm:agents:architect "set personality to civictech technical visionary..."
```

### 3. Run Your First Workflow
```bash
/bmad:bmm:workflows:brainstorming-session "our strategic priorities for next quarter"
```
